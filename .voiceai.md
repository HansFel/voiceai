# VoiceAI – Helpdesk-Kontext

## Was ist VoiceAI?
VoiceAI ist ein KI-gestützter Assistent mit Spracheingabe und -ausgabe.
Er läuft als Docker-Container und ist über den Browser erreichbar.
Administratoren können ihn als Helpdesk für andere Software-Instanzen konfigurieren.

## Funktionen
- **Chat** — Textbasierter Chat mit KI (Anthropic Claude oder Mistral)
- **Spracheingabe** — Mikrofon-Button, sprich deine Frage
- **Sprachausgabe** — Antworten werden vorgelesen (🔊 Button zum Ein/Ausschalten)
- **Agent-Modus** — Zugriff auf Code-Repositories (nur für Entwickler)
- **Helpdesk-Modus** — Beantwortet Fragen zu einer bestimmten Software anhand der Dokumentation

## Modell wählen
Das zuletzt verwendete Modell wird gespeichert. Verfügbare Modelle hängen von den konfigurierten API-Keys ab:
- **Claude Sonnet / Opus** (Anthropic) — sehr leistungsfähig
- **Mistral Large / Small** — gute Alternative, schnell

## Helpdesk-Modus (nur Admin/Developer)
1. **📋 Helpdesk** Button klicken
2. Repo aus der Liste wählen (Repos mit `.voiceai.md` werden angezeigt)
3. Fragen zur jeweiligen Software stellen — die KI antwortet anhand der Dokumentation
4. "✕ Helpdesk aus" zum Deaktivieren

## Agent-Modus (nur Developer)
🤖 Agent Button aktivieren → KI kann Repositories lesen, Git-Status prüfen, Code analysieren.

## Benutzerverwaltung (nur Admin)
⚙ Admin Link oben rechts → Benutzer anlegen, Rollen vergeben, Passwörter zurücksetzen.

## Login
Email eingeben → Weiter → Passwort eingeben.
Erstes Login: Passwort selbst festlegen (mind. 6 Zeichen).
Bei vergessenem Passwort: Administrator kontaktieren → PW Reset.

## Rollen
- **user** — nur Chat/Helpdesk, kein Agent, kein Admin
- **developer** — Chat + Agent-Modus + Helpdesk-Toggle
- **admin** — alles + Benutzerverwaltung
