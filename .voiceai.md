# VoiceAI – Projekt-Kontext für Code-Agent

## Architektur
- **Backend**: Python/Flask + Flask-SocketIO (`backend/app.py`) — async_mode='threading'
- **Frontend**: Einzelne HTML-Datei (`frontend/index.html`) — kein Framework, vanilla JS
- **Docker**: `docker-compose.yml` — Port 5010→5000, Volumes für frontend/ und backend/ (live-reload)
- **Traefik**: Route `/wsl-voiceai` mit Strip-Prefix, `/socket.io` als eigener Router

## Bereits implementierte Features
- Chat (Anthropic Claude + Mistral) mit Modell-Auswahl
- Socket.IO Echtzeit-Kommunikation
- Spracheingabe (Web Speech API) + Sprachausgabe (TTS)
- Login/Auth mit Email + Passwort, Rollen: user / developer / admin
- Admin-Dashboard: Benutzer einladen, Rollen vergeben, PW-Reset
- Helpdesk-Modus: Repo mit `.voiceai.md` wählen → KI antwortet anhand Doku
- Agent-Modus: KI kann Repos lesen, Git-Status prüfen
- **Code-Agent**: Direkte Dateiänderungen via list_files / read_file / write_file / run_command
- **Coding-Modus**: 💻-Button → Repo-Auswahl → jede Nachricht geht direkt in Code-Agent (kein Confirm)
- Dev-Intent-Erkennung + gelbe Bestätigungsleiste (im Normal-Modus für Developer)
- Provider wählbar: Anthropic (claude-sonnet-4-6) oder Mistral (mistral-large-latest)

## API-Routen (backend/app.py)
- `GET /api/me` — eingeloggter User (role, email, helpdesk_mode, helpdesk_repo)
- `GET /api/models` — verfügbare Modelle je nach API-Keys
- `GET /api/all-repos` — alle Verzeichnisse unter /repos/
- `GET /api/helpdesk-repos` — Repos mit .voiceai.md
- `POST /api/helpdesk-mode` — Helpdesk ein/aus + Repo setzen
- `POST /api/chat` — (nicht verwendet, Chat läuft über Socket.IO)
- `GET /admin` — Admin-Dashboard

## Socket.IO Events
- `chat` → `response` — normaler Chat
- `agent` → `response` — Agent-Modus mit Repo-Zugriff
- `code_agent` → `code_agent_output` + `code_agent_done` — Code-Agent
- `check_dev_intent` → `dev_intent_result` — Intent-Erkennung

## Wichtige Konventionen
- Alle UI-Änderungen in `frontend/index.html` (eine Datei, kein Build-Schritt)
- Backend-Änderungen in `backend/app.py` — nach Änderung `docker restart voiceai` nötig
- Frontend wird live aus dem gemounteten Volume serviert — kein Neustart nötig
- Kein TypeScript, kein React, kein Build-Tool
- CSS ist inline im `<style>` Block der index.html
- Farbschema: dunkel (#0f0f0f Hintergrund, #1a1a1a Cards, #2563eb Akzent)
- Coding-Modus: orange (#b45309), Helpdesk: cyan (#0e7490), Agent: lila (#7c3aed)

## Umgebungsvariablen
- `ANTHROPIC_API_KEY` — für Claude
- `MISTRAL_API_KEY` — für Mistral
- `REPOS_BASE=/repos` — Basis-Pfad für alle Repos
- `APP_URL` — öffentliche URL (für Session-Cookie und Redirects)
- `SECRET_KEY` — Flask Session Secret
